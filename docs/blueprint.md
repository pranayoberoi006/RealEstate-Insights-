# **App Name**: RealEstate Insights

## Core Features:

- Property Search: Allows users to search for properties based on location, category (house, studio, shop), and number of bedrooms (3BHK, 2BHK, etc.).
- Price Prediction Tool: Uses machine learning model trained on available datasets to provide estimated price prediction, offering insights into potential property values in different locations.
- Interactive Map: Displays property locations on an interactive map with visual representation of price ranges and property categories.
- User Authentication: Enables users to create accounts, log in, and save their search preferences.
- Header Navigation: Provides easy navigation to different sections of the website, including Home, Team Behind This, About Us, and How It Works.
- Detailed Property View: Displays comprehensive information for each property, including location, property category, features, and the AI-powered price prediction.
- Insights Explanation: A How It Works explanation with generative AI "tool" used to tailor advice and suggestions for how the service can provide them value.

## Style Guidelines:

- Primary color: Deep blue (#34495E) to convey trust and professionalism, suitable for a finance-related service.
- Background color: Light gray (#F4F5F6) for a clean and modern look.
- Accent color: Teal (#2ECC71) to highlight key information and calls to action.
- Body font: 'PT Sans', a humanist sans-serif.
- Headline font: 'Playfair', a modern serif. If longer text is anticipated, use 'PT Sans' for body text with the 'Playfair' headline font.
- Use minimalist, professional icons for property categories, features, and navigation.
- Employ a clean and spacious layout with clear information hierarchy, ensuring ease of use and visual appeal.
